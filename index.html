to national scale.
Production-ready stack
Frontend
📱
React Native

Single codebase → iOS + Android. Reusable kid-optimised component library.

🌐
React (Web Admin)

Admin panel + parent dashboard as responsive web — no separate build cycle.

Backend & Data
⚡
Node.js + PostgreSQL

REST + WebSocket for live tournaments. Horizontal scaling with Redis for leaderboards.

☁️
AWS / GCP

CDN for audio assets, auto-scaling groups for competition peak traffic spikes.

Difficulty adaptation, smart word recommendations, personalised learning path generation.

📊
Mixpanel + Cohort

Funnel tracking, streak analytics, churn prediction, retention cohort analysis.

All student data encrypted at rest and in transit. COPPA and DPDP (India) compliant by design. Parental consent flows built into onboarding.
